# Welcome

## 第一个Java程序

```java
//注释有三种 // /**/ /***/
//主要说下第三种/***/ 该注释可生成文档
public class FirstSamlie{
    public static main(String[] args){
        //System.out.println 换行输出 System.out.print 不换行输出
        System.out.println("We will not use 'Hello World'");
    }
}
```

## 注意

* java区分大小写
* 源文件名必须与类名相同
* 以上就是一个简单的java模板，其他的东西暂时不细说
* 强类型语言，数据类型必须有声明
